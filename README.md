# NetflixDatabase-

¡Hola! Bienvenido a mi primer repositorio de la serie: "Subiendo proyectos hasta conseguir trabajo".

Este es mi primer proyecto enfocado en SQL, el lenguaje con el que inicié mi camino en el desarrollo de software. El proyecto consiste en el diseño e implementación de una base de datos relacional para la gestión de contenido (series y episodios) inspirado en plataformas de streaming como Netflix.

🎯 Propósito del Proyecto:
Demostrar mis capacidades técnicas en el manejo de bases de datos mediante la resolución de problemas reales, tales como:

* Modelado de datos: Creación de diagramas Entidad-Relación (DER) profesionales.

* Consultas complejas: Uso de JOINs, filtros avanzados y ordenamiento de datos.

* Normalización: Gestión eficiente de relaciones entre series, episodios y actores.

* Diseño de Base de Datos: Implementación de Claves Primarias (PK), Claves Foráneas (FK), manejo de Cardinalidad y principios de Normalización.

* Gestión de Datos: Creación y mantenimiento de tablas mediante comandos DDL (Definición), DML (Manipulación) y operaciones CRUD.

🛠️ Herramientas utilizadas:

MySQL Workbench: Para el modelado, ingeniería inversa y ejecución de scripts.
